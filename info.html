About This Weather App

This Weather App is designed by me, Tanmay Khambete(23BCE0498) to provide users with real-time weather updates and detailed forecasts in a clean, intuitive, and responsive interface. Whether you're planning your day or checking air quality levels, this app gives you all the essential information in one place. It also uses a responsive design so that the website works on any device no matter the width.

Features include:

  • Live weather data by city or current location
  • Air quality index with easy-to-read color coding
  • Hourly and weekly forecasts
  • Sunrise and sunset times
  • Details about Humidty, Pressure, Visibility, Wind Speed and Feels Like
  • Responsive layout for all devices

Built with: HTML, CSS, JavaScript, and OpenWeatherMap API.

Created as a project for the course Web Programming (BCSE203E). You can view the source code on GitHub.
